[Image Manager](https://github.com/davidvkimball/obsidian-image-manager) allows you to quickly drag and drop image files, insert images with a command, or paste images directly from your clipboard.

### Features

- **Remote image sources** — pull in images from Unsplash, Pexels, or Pixabay with just a few keystrokes.
- **SEO-friendly** — immediately set alt text and a file name when inserting remote images.
- **Banner images** — display a banner on top of any Markdown file when a valid `image` property is defined. Matches the cover image used in the [Bases CMS](/plugins/bases-cms/) view by default.

### Commands

- **Insert remote image**: Search and insert an image from Unsplash, Pexels, or Pixabay. Activate with `CTRL + '`.
- **Insert into property**: Insert an image into the designated property. Activate with `CTRL + SHIFT + '`.
